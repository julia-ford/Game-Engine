Detailed Description

One of the most low-level classes in the engine, "Thing" represents any object that can be interacted with. All other interactable classes, including Person, derive from Thing.

void GameEngineJuly2013.Code.Thing.Move ( Direction  dir)

This has the thing put in a request to the GameManager to be moved in a given direction. This function does not handle the actual movement; GameManager.Move(Thing, Direction) does that. However, this function does do error checking. If there is no room in the specified direction, the function throws an exception and no movement request is made.
