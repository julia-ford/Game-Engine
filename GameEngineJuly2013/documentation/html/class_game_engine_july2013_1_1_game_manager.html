static void GameEngineJuly2013.GameManager.ReportIfVisible ( Thing  item,
string  message 
)
static

This is a stub for now, but the idea is that things will send requests to the GameManager to print their messages, which are generated when actors do things or events take place. The GameManager will decide if the player could have observed the event, based on where the player is as compared to where the thing sending the messages is. If the player could reasonably observe the event, then the message is printed. If not, the message is discarded.
